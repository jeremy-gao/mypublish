# 功能介绍
---------------
 - 支持页面SVN UPDATE
 - 支持排除提交,选择特定文件提交
 - 支持对比目录文件差异，支持对比文件内容详细差异
 - .....

# 安装部署
---------------
 - 安装pytho2.7,python-pip
 - pip install bottle ConfigParser beaker
 - yum -y install httpd mod_wsgi git subversion
 - 修改config.ini配置文件
